TrID_Identifier  1.0

Контентный плагин для TC, определяет тип файла с помощью утилиты TrID (http://mark0.net/soft-trid-e.html).
Это новая независимая версия плагина TrID (http://wincmd.ru/plugring/trid.html).


*** Установка ***

При попытке открыть архив с плагином TC выдаст сообщение об установке плагина.
Скопировать утилиту TrID и файл triddefs.trd в подкаталог TrID\ плагина.
Создать новый набор колонок (Конфигурация - Настройка - Наборы колонок - Новый).
Добавить новые колонки с полями из "trid_identifier".
