LinkInfo 1.52
  Это объединённый проект, состоящий из двух частей:
1. LinkEditor - Программа для редактирования LNK-файлов
2. LinkInfo   - Lister-плагин для просмотра и редактирования их из TC

Автор: Власов Константин, 2011 г.
Домашняя страница: http://flint-inc.ru/
E-mail:            support@flint-inc.ru


    /===========================\
    |      История версий:      |
    \===========================/

Версия 1.52:
  1. Плагин был переписан с добавлением полной поддержки юникода.
  2. Добавлена 64-битная версия.
  3. Визуальное представление в окне просмотра значительно улучшено.
  4. Добавлена поддержка переноса строк по правой границе окна просмотрщика.
  5. Добавлен файл автоустановки.
  6. Несколько других, незначительных улучшений.

Версия 1.51:
  1. Исправлен очень неприятный и неуловимый (некогда) глюк: на некоторых компьютерах
     при закрытии окна Lister'а закрывался и сам Total Commander.

Версия 1.5:
  1. Иконка в окне Lister'а теперь показывается для абсолютно всех ярлыков.
  2. В диалоге редактирования при выборе иконки по умолчанию выбрана именно та иконка,
     которая отображается (не всегда, но в большинстве случаев). В силу ограничений Win9x,
     в этих системах эта возможность работает реже.
  3. Поправлен глюк с сохранением ярлыков на системные объекты.
  4. Теперь у ярлыков на системные объекты отображается правильное имя.

Версия 1.42:
  1. Иконка теперь рисуется на том же фоне, что и фон окна RichEdit (раньше рисовалась только на белом).
  2. Добавлена возможность вызова кнопок "Сохранить", "Сохранить как..." и "Закрыть" в диалоге клавишами
    (буквы вызова подчёркиваются, как в меню).
  3. Иконка для ярлыка на папку теперь показывается.

Версия 1.41:
  1. Поправлен баг, возникающий на всех системах, кроме WinXP и Win2003.

Версия 1.4:
  1. Диалог редактирования теперь может изменять размер, который (вместе с положением на экране)
     восстанавливается при следующем запуске (при вызове из просмотрщика восстанавливается только
     размер, положение задаётся местом вызова контекстного меню).
  2. Иконка отображается не только в окне Lister'а, но и в диалоге.
  3. В поле для номера иконки в диалоге редактирования добавлены стрелочки.
  4. Теперь иконка в окне просмотра является не отдельной нашлёпкой, а полноправным объектом
     RichEdit - её можно копировать в буфер, перетаскивать мышкой и т.д.
  5. Задавлен баг, возникающий при одновременном просмотре нескольких файлов.
  6. Поправлено отображение горячей клавиши в окне Lister'а (раньше показывались правильно только
     буквенно-цифровые сочетания).
  7. Куча мелких фиксов.

Версия 1.31:
  1. Появился просмотр иконки файла в виде картинки. На данный момент эта картинка не является
     частью собственно документа, её нельзя выделить, скопировать и т.д. В будущем такая
     возможность скорее всего появится, но только если это действительно кому-то надо.
  2. Улучшена совместимость с Win9x - в частности диалог выбора иконок работал под Win9x не
     совсем правильно (выдавал не тот стартовый путь, вместо выбранного пути выдавал кракозяблы...)

Версия 1.3:
  1. Исправлен злостный баг, который я так долго не мог поймать - на многих компьютерах
     плагин не работал и не поддавался никакому перевоспитанию.
  2. При нажатии клавиши ContextMenu на клавиатуре меню теперь появляется посередине окна
     просмотрщика.
  3. Диалог редактирования появляется в том месте, где находился курсор мышки в момент вызова
     контекстного меню (если вызов меню был произведён не с клавиатуры).
  4. В диалоге выбора каталога теперь автоматически подставляется указанный каталог
     (раньше был "Мой компьютер").
  5. Подкорректированы размеры надписей на диалоге выбора папки.
  6. У всех диалогов, появляющихся при нажатии "..." теперь установлены стартовые пути на
     содержимое соответствующих полей ввода.

Версия 1.21:
  1. Ликвидировано падение программы/плагина при вызове диалога выбора значка.
  2. Поправлено отображение номера значка.

Версия 1.2:
  1. В этой версии наконец, произведена интеграция обеих частей программы в единый проект.
     В плагине добавлена возможность редактировать просматриваемый файл путём выбора
     "Редактировать..." из контекстного меню. После сохранения данных диалог закрывается
     и окно просмотра автоматически обновляется.
  2. В диалогах редактирования активирована кнопка выбора значка.
  3. Добавлен манифест WinXP для лучшего отображения в системе Windows XP.
  4. Переименованы составные части проекта.

Версия 1.1:
  Следующая версия. В программу добавлена возможность редактировать существующие ссылки.
  Теперь когда пользователь открывает LNK-файл, все поля диалога автоматически заполняются
  данными об открытой ссылке.

Версия 1.0:
  Первая версия программы, позволяющая только создавать LNK-файлы с указанной пользователем
  информации:
  1. Имя целевого файла
  2. Аргументы командной строки
  3. Рабочий каталог приложения
  4. Описание ссылки
  5. Горячую клавишу
  6. Файл, содержащий значок для этой ссылки, и номер значка в этом файле
  7. Размер окна запускаемого приложения
  Первая версия плагина позволяет просматривать всю вышеуказанную информацию.
