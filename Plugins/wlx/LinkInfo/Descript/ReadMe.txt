LinkInfo 1.52
  Это объединённый проект, состоящий из двух частей:
1. LinkEditor - Программа для редактирования LNK-файлов
2. LinkInfo   - Lister-плагин для просмотра и редактирования их из TC

Автор: Власов Константин, 2011 г.
Домашняя страница: http://flint-inc.ru/
E-mail:            support@flint-inc.ru


Описание
--------
Плагин позволяет просматривать LNK-файлы в Total Commander простым нажатием F3.
При этом в окне просмотрщика выводится следующая информация:
1. Целевой объект ссылки
2. Дополнительные аргументы для запуска приложения
3. Рабочий каталог программы
4. Описание
5. Горячая клавиша
6. Размер окна
7. Значок (путь к файлу со значком и её номер в этом файле)

Нажатие правой кнопкой в окне просмотрщика выводит контекстное меню с командой
"Редактировать...", которая открывает диалог редактирования ярлыка, позволяющий
поменять значение любого из полей, отображающихся в окне просмотра.

Также в комплекте с плагином идёт отдельная программа для редактирования
LNK-файлов. При её запуске выводится то же самое диалоговое окно редактирования
ярлыка, причём путь к открываемому файлу можно задать аргументом командной строки.

Эти две части проекта независимы и могут работать по отдельности. Нет необходимости
держать программу LinkEditor в одной папке с LinkInfo.


Системные требования
--------------------
32-битная версия:
	Windows 2000 и выше
	Total Commander 7.50–8.0 x32

64-битная версия:
	Процессор с поддержкой 64-битных команд (Intel Pentium 4, AMD Athlon 64 и выше)
	Windows XP x64 и выше
	Total Commander 8.0 x64
