Инструкция по установке:
1. Копировать файл maultosar.bat на диск с установленной системой (Например, C:/);
2. Внести изменения в реестр запустив файл реестра maultosar.reg
3. Из папки Key запустить файл Patch.exe и пропатчить файл Totalcmd или Totalcmd64 (Можно и два пропатчить)
4. Из папки Key перенести файл wincmd.key в корневой каталог Total Commander
